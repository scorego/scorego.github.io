Redis(一) Redis简介

发表于 2019-05-13 | 更新于 2019-05-15 | 分类于 NoSQL

Redis(Remote dictionary server) 是一个开源的、使用ANSI C语言编写的内存数据结构存储系统，可用作数据库、缓存和消息代理。Redis次版本号(第一个小数点后的数字)为偶数表示是稳定版，基数表示是不稳定版。

Redis官网
Redis命令参考

Redis的优点如下：

  • 速度快，因为数据存在内存中。
  • 支持丰富数据类型。
  • 支持事务，操作都是原子性。
  • 丰富的特性。
  • 支持主从复制。
  • 支持数据持久化。

发展历史：

  • 2009，初始发布
  • 2010，VMware开始赞助
  • 2015，Redis 3.0
  • 2018.10， Redis 5.0 GA

安装：

  1. POSIX系统

  2. OS X
    可以通过Homebrew和MaxPorts等安装。

  3. Windows
    Redis官方不支持windows。非官方版本：https://github.com/microsoftarchive/redis
