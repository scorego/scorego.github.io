Java并发02 基本概念

发表于 2019-06-02 | 更新于 2019-06-03 | 分类于 Java并发

1. Java线程状态

Java Language Specification : Chapter 17. Threads and Locks

线程可以有以下6种状态：

  1. New / 新建
  2. Runnable / 可运行
  3. Blocked / 被阻塞
  4. Waiting / 等待
  5. Timed waiting / 计时等待
  6. Terminated / 终止


线程状态涉及到有Object、Thread和Synchronized关键字等。Object类有notify()、notifyAll()、wait()方法，Thread类定义了一些sleep()、yield()、interrupt()、join()等方法。

2. 常用实现多线程的两种方式

A. Runnable接口

实现Runnable接口的类的实例是将在一个线程执行的，其必须实现无参方法run()。Runnable接口可以实现无需继承Thread类来新建线程。

B. Thread类

start()方法的作用是启动线程，不能被重复调用。调用start()方法并不能保证线程会立刻执行。run()就和普通的成员方法一样，可以被重复调用。单独调用run()的话，会在当前线程中执行run()，而并不会启动新线程。
