1. FileInputStream

FileInputStream类使得可以将文件的内容作为字节流读取。该类是InputStream的子类，意味着可以将FileInputStream用作InputStream。FileInputStream用来读取原始字节，比如图片数据。当读取字符流时，考虑使用FileReader。

2. FileOutputStream

FileOutputStream用来向一个File或FileDescriptor写入字节流。有的平台只允许一个文件同时被一个FileOutputStream打开，此时对改文件创建新的FileOutputStream的构造函数会返回失败。FileOutputStream用来写入原始字节，比如图片数据。如果要写入字符，考虑使用FileReader。
