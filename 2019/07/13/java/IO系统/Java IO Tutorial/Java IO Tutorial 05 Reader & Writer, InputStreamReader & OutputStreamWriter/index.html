Java IO Tutorial 05 Reader & Writer, InputStreamReader & OutputStreamWriter

发表于 2019-07-13 | 更新于 2019-07-15 | 分类于 Java IO

1. Reader

Reader类(java.io.Reader)是Java IO API中所有Reader子类的基类。Reader类似于InputStream，只是Reader是基于字符而不是基于字节的。换句话说，Reader用于读取文本，而InputStream用于读取原始字节。
许多应用程序使用UTF(UTF-8或UTF-16)来存储文本数据。UTF-8中，每个字符可能需要一个或多个字节来表示；而在UTF-16中，每个字符需要2个字节来表示。因此，在读取文本数据时，数据中的单个字节可能与UTF中的一个字符不对应。如果只是通过InputStream每次读取UTF-8数据的一个字节，并尝试将其转换为char，则可能不会得到期望的文本。Reader类正是解决这个问题，其能够将字节转换成字符，只需要告诉Reader要解码的字符集。这是在实例化Reader类(实际上是实例化其一个子类)时完成的。
